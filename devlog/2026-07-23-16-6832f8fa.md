# Security Update: GitHub Cuts Public Bug Bounty Payouts, Moves Top Rewards to VIP Tier

_2026-07-23 16:13 UTC_

[Source](https://thehackernews.com/2026/07/github-cuts-public-bug-bounty-payouts.html)

**Devlog Entry: GitHub Security Update**
====================================================

### Date: 23 July 2026

GitHub has announced a change to its bug bounty program, reducing public payouts and reserving top rewards for its VIP tier members. This shift may incentivize more hackers to focus on private, sponsored bug hunting, potentially reducing the pool of publicly reported vulnerabilities.
