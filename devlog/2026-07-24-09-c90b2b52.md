# Security Update: Kimi K3 Agents Found Redis Zero-Days and Built RCE Exploit, Researchers Say

_2026-07-24 09:14 UTC_

[Source](https://thehackernews.com/2026/07/kimi-k3-agents-found-redis-zero-days.html)

**Devlog Entry: Redis Zero-Days and Kimi K3 Agents**
=====================================================

**Date:** 24 July 2026

**Summary:** Researchers have discovered zero-day vulnerabilities in Redis, an in-memory data store popularly used in web applications. The Kimi K3 agents have allegedly exploited these vulnerabilities to build a remote code execution (RCE) exploit, highlighting the potential for severe attacks on Redis-powered systems.

**Recommendations:**

* Review and update Redis configurations to ensure secure access and authentication
* Implement regular security audits and vulnerability scans to detect potential exploits
* Consider upgrading to the latest Redis version to mitigate known vulnerabilities
