# Security Update: Bing Images Flaws Let Crafted SVGs Run Commands as SYSTEM on Microsoft's Servers

_2026-07-24 13:35 UTC_

[Source](https://thehackernews.com/2026/07/bing-images-flaws-let-crafted-svgs-run.html)

**Devlog Entry: Vulnerability in Bing Images**

**Date:** 2026-07-24

**Summary:** A newly discovered vulnerability in Bing Images allows maliciously crafted SVGs to execute commands with SYSTEM privileges on Microsoft's servers. This flaw allows attackers to run arbitrary code on Microsoft's servers, potentially leading to data theft or further exploitation. The issue highlights the importance of robust input validation and sanitization in web applications.
